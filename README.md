# C Program Codes

A collection of C programming projects and practice exercises covering fundamental to intermediate programming concepts.

## 📋 Contents

### Core Programs
- **1-10 prime num.c** - Find prime numbers from 1 to 10
- **Armstrong number.c** - Check if a number is an Armstrong number
- **prime numbers.c** - Prime number generation and checking
- **Multiplication table.c** - Generate multiplication tables
- **swap two num.c** - Swap two numbers using different methods
- **reverse of the number.c** - Reverse a given number
- **fibbancci series.c** - Generate Fibonacci sequence
- **Dynamic memeory allocation.c** - Dynamic memory allocation examples
- **double pointer.c** - Double pointer operations and demonstrations
- **Identical strings.c** - Compare and check identical strings
- **Elimate vowels.c** - Remove vowels from a string

### Matrix Operations
- **addition of matrics.c** - Add two matrices
- **largest num in matrics.c** - Find the largest number in a matrix
- **mulitply the matrics.c** - Multiply two matrices
- **diagonal elements.c** - Extract and work with diagonal elements


### Output Files
- `output/` - Compiled executable files (.exe)
- `c pratice/output/` - Practice program executables
- `exp 8/output/` - Experiment executable files

## 🛠️ Requirements

- GCC Compiler (MinGW on Windows)
- C Standard Library (stdio.h, stdlib.h, string.h, math.h, etc.)
- Any text editor or IDE (Visual Studio Code, Dev-C++, etc.)

## 💻 How to Compile and Run

### On Windows (using GCC):

1. Open Command Prompt or PowerShell
2. Navigate to the directory containing the .c file:
   ```bash
   cd "path\to\C Program codes"
   ```

3. Compile the program:
   ```bash
   gcc -o output_filename program_name.c
   ```
   Example:
   ```bash
   gcc -o output\prime prime numbers.c
   ```

4. Run the compiled program:
   ```bash
   output\output_filename.exe
   ```
   Example:
   ```bash
   output\prime.exe
   ```

### On Linux/macOS:

1. Open Terminal
2. Navigate to the directory:
   ```bash
   cd path/to/C\ Program\ codes
   ```

3. Compile:
   ```bash
   gcc -o output_filename program_name.c
   ```

4. Run:
   ```bash
   ./output_filename
   ```
